## DICAS DE FILMAGEM

### Equipamento Mínimo
```
- Celular com boa câmera (iPhone 12+ ou similar)
- Tripé ou apoio estável
- Iluminação natural (janela) ou ring light
- Microfone de lapela (melhora MUITO o áudio)
- Ambiente silencioso
```

### Enquadramento
```
- Você no centro ou regra dos terços
- Fundo limpo (parede, estante organizada, plantas)
- Espaço acima da cabeça, mas não muito
- Olhe para a câmera, não para a tela
```

### Gravação de Tela
```
- Use OBS ou Loom para gravar
- Resolução: 1920x1080 mínimo
- Limpe abas do navegador antes
- Feche notificações
- Mouse movements suaves e intencionais
- Zoom em áreas importantes
```

### Edição
```
- Cortes rápidos entre cenas (mantém atenção)
- Música de fundo sutil (royalty-free)
- Texto na tela para pontos-chave
- Logo no canto
- Legenda (muitos assistem sem som)
```

## O QUE MOSTRAR NA DEMO

### Sequência Recomendada

```
1. Página Inicial
   - Design moderno
   - Estatísticas da plataforma
   - Menu de navegação

2. Lista de Propriedades
   - Cards com informações
   - Filtros (se tiver)
   - Yield, valor, progresso de venda

3. Detalhe de Propriedade
   - Informações completas
   - Gráficos (se tiver)
   - Botão de investir

4. Conexão de Wallet
   - Modal de wallets
   - Phantom conectando
   - Endereço aparecendo

5. Página de KYC (se aplicável)
   - Processo de verificação
   - Status da credencial

6. Portfolio
   - Tokens que possui
   - Valor total
   - Dividendos recebidos

7. Documentação
   - Sidebar com seções
   - Diagramas Mermaid
   - Código de exemplo

8. Admin (opcional)
   - Mostrar que existe
   - Criação de propriedade
```

### O que NÃO mostrar
```
- Erros ou bugs
- Páginas em construção
- Dados sensíveis (chaves, senhas)
- URLs internas de desenvolvimento
- Console do navegador com erros
```
